# ✨ CHR birth 직소 퍼즐 이벤트 ✨

이 프로젝트는 웹 브라우저에서 가볍게 즐길 수 있는 **이벤트용 직소 퍼즐 게임**입니다. 
모바일과 PC 환경을 모두 지원하며, `headbreaker` 라이브러리를 기반으로 제작되었습니다.

---

## 📂 올바른 폴더 구조 (매우 중요!)

퍼즐 게임이 정상적으로 이미지를 불러오려면 반드시 아래와 같은 구조로 파일이 배치되어야 합니다.

```text
CHR birth/ (가장 바깥 폴더)
├── index.html (게임 실행 파일)
└── images/ (폴더 이름은 반드시 소문자)
    └── puzzle.jpg (퍼즐에 사용할 정사각형 이미지)
